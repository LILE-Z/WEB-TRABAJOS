Actividad 11 (Entregar).

Investigar más a fondo el tema de las Media Queries y realizar una página con algunos elementos HTML, los cuales puedan ser transformados haciendo uso de más Media Queries.

Este es un ejemplo de cómo usar media queries en CSS para hacer un diseño web responsivo. Las media queries pueden usarse para cambiar colores, mostrar u ocultar elementos, modificar la disposición de la cuadrícula, y mucho más, dependiendo de las características del dispositivo.
